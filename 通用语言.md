# 通用语言 (Ubiquitous Language)

> 本文档定义了本项目的领域通用语言，包含中文术语和英文术语对照。
> 所有团队成员和代码中的命名应当以此为准。

---

## 分层架构 (Layered Architecture)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 领域层 | Domain Layer | 核心业务逻辑，不依赖任何其他层 |
| 应用层 | Application Layer | 用例编排，协调领域对象完成用户请求 |
| 基础设施层 | Infrastructure Layer | 技术实现，如持久化、外部服务 |
| 表现层 | Presentation Layer | 用户界面，Vue 组件和 Store |

---

## 领域实体 (Domain Entities)

### 文档 (Document) — 聚合根

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 文档 | Document | 聚合根，包含一组问题和一段对话 |
| 文档 ID | Document.id | 唯一标识 |
| 文档标题 | Document.title | 文档名称，不能为空 |
| 文档问题列表 | Document.questions | 文档关联的 Order{Question} |
| 文档对话 | Document.conversation | 文档关联的 Conversation |
| 问题数量 | Document.questionCount | 问题的数量 |
| 更新标题 | Document.updateTitle() | 修改文档标题 |
| 添加问题 | Document.addQuestion() | 向文档新增一个问题 |
| 移除问题 | Document.removeQuestion() | 从文档中删除指定问题 |
| 重排序问题 | Document.reorderQuestions() | 按指定 ID 顺序重新排列问题 |
| 获取摘要 | Document.getSummary() | 获取文档的概要信息 |
| 序列化 | Document.toJSON() | 将文档转为 JSON 格式 |
| 文档摘要 | DocumentSummary | 文档的轻量摘要（id, title, questionCount, messageCount） |

### 问题 (Question)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 问题 | Question | 用户提出的一个问题，属于某个文档 |
| 问题 ID | Question.id | 唯一标识 |
| 问题文本 | Question.text | 问题的内容文字 |
| 问题序号 | Question.order | 在文档中的排序位置 |
| 更新文本 | Question.updateText() | 修改问题内容 |
| 变更序号 | Question.changeOrder() | 修改排序位置 |
| 序列化 | Question.toJSON() | 将问题转为 JSON 格式 |

### 对话 (Conversation)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 对话 | Conversation | 围绕文档的问答消息序列 |
| 对话 ID | Conversation.id | 唯一标识 |
| 消息列表 | Conversation.messages | 对话中的所有消息 |
| 消息数量 | Conversation.messageCount | 消息的总数 |
| 添加消息 | Conversation.addMessage() | 向对话追加一条消息 |
| 移除消息 | Conversation.removeMessage() | 从对话中删除指定消息 |
| 根据 ID 获取消息 | Conversation.getMessageById() | 通过消息 ID 查找消息 |
| 获取问答对 | Conversation.getQuestionAnswerPair() | 获取问题和对应的回答配对 |
| 获取最后一条消息 | Conversation.getLastMessage() | 获取对话中最新一条消息 |
| 序列化 | Conversation.toJSON() | 将对话转为 JSON 格式 |

### 消息 (Message)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 消息 | Message | 对话中的单条消息，可以是问题或回答 |
| 消息 ID | Message.id | 唯一标识 |
| 消息类型 | Message.type | 取值 question（提问）或 answer（回答） |
| 消息内容 | Message.content | 消息的正文 |
| 关联问题 ID | Message.questionId | 所关联问题的 ID |
| 创建时间 | Message.createdAt | 消息创建的时间戳 |
| 编辑时间 | Message.editedAt | 消息最后编辑的时间戳 |
| 是否已编辑 | Message.isEdited | 标记消息是否被编辑过 |
| 编辑内容 | Message.editContent() | 修改消息内容 |
| 序列化 | Message.toJSON() | 将消息转为 JSON 格式 |
| 消息类型 | MessageType | 类型字面量：'question' \| 'answer' |

---

## 问答关系 (Question-Answer Relationship)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 回答 | Answer | 对某个问题的回复，是一条 type='answer' 的消息 |
| 问答对 | Question-Answer Pair | 一个问题及其对应的回答组成的配对 |
| 关联问题 | Linked Question | 回答通过 questionId 关联到对应的问题 |
| 获取问答对 | Conversation.getQuestionAnswerPair() | 按问题 ID 获取该问题对应的问答对 |
| 根据问题 ID 获取消息 | Conversation.getMessagesByQuestionId() | 按问题 ID 查找所有关联的提问和回答消息 |

## 领域错误 (Domain Errors)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 领域错误 | DomainError | 领域层异常的基类 |
| 验证错误 | ValidationError | 输入验证失败时抛出（如标题为空、序号无效） |
| 未找到错误 | NotFoundError | 实体未找到时抛出（如文档/问题/消息不存在） |

## 领域事件 (Domain Events)

### 文档事件

| 中文术语 | 英文术语 | 触发时机 |
|---------|---------|---------|
| 文档已选择 | DocumentSelectedEvent | 用户选中某个文档时 |
| 文档列表已加载 | DocumentsLoadedEvent | 文档列表加载完成后 |
| 文档已创建 | DocumentCreatedEvent | 新文档创建成功后 |
| 文档已更新 | DocumentUpdatedEvent | 文档属性（如标题）更新后 |
| 文档已删除 | DocumentDeletedEvent | 文档被删除后 |

### 问题事件

| 中文术语 | 英文术语 | 触发时机 |
|---------|---------|---------|
| 问题已激活 | QuestionActivatedEvent | 用户点击选中某个问题时 |
| 问题已添加 | QuestionAddedEvent | 文档新增问题后 |
| 问题已移除 | QuestionRemovedEvent | 文档移除问题后 |
| 问题已重排序 | QuestionsReorderedEvent | 文档问题顺序调整后 |

### 设置事件

| 中文术语 | 英文术语 | 触发时机 |
|---------|---------|---------|
| 主题已切换 | ThemeToggledEvent | 深色/浅色主题切换后 |
| 设置已变更 | SettingsChangedEvent | 任意设置项更新后 |

### 对话事件

| 中文术语 | 英文术语 | 触发时机 |
|---------|---------|---------|
| 消息已发送 | MessageSentEvent | 新消息（提问或回答）发送后 |
| 消息已编辑 | MessageEditedEvent | 消息内容修改后 |
| 消息已删除 | MessageDeletedEvent | 消息删除后 |

---

## 事件总线 (Event Bus)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 事件总线 | EventBus | 领域事件的发布-订阅接口 |
| 简单事件总线 | SimpleEventBus | EventBus 的默认实现，支持内存中的发布/订阅 |
| 全局事件总线 | globalEventBus | 应用全局单例的事件总线实例 |
| 订阅 | EventBus.on() | 注册指定类型的事件处理器 |
| 取消订阅 | EventBus.off() | 移除指定类型的事件处理器 |
| 一次性订阅 | EventBus.once() | 只响应一次的事件处理器 |
| 发布 | EventBus.emit() | 发布一个领域事件 |
| 清除 | EventBus.clear() | 移除所有事件处理器 |

## 仓库 (Repositories)

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 文档仓库 | DocumentRepository | Document 聚合根的存储接口 |
| 设置仓库 | SettingsRepository | 应用设置的存储接口 |
| 设置 | Settings | 应用配置，含 darkMode 等字段 |
| 查找全部 | DocumentRepository.findAll() | 获取所有文档 |
| 根据 ID 查找 | DocumentRepository.findById() | 按 ID 查找文档 |
| 保存 | DocumentRepository.save() | 保存或更新文档 |
| 删除 | DocumentRepository.delete() | 删除指定文档 |
| 是否存在 | DocumentRepository.exists() | 检查文档是否存在 |
| 获取设置 | SettingsRepository.getSettings() | 获取全部设置 |
| 保存设置 | SettingsRepository.saveSettings() | 保存全部设置 |
| 获取项 | SettingsRepository.getItem() | 获取单个设置值 |
| 设置项 | SettingsRepository.setItem() | 设置单个配置项 |
