# 问题列表规范

## 新增需求

### 需求：问题列表展示

右侧栏应展示当前文档的问题列表。

#### 场景：问题列表显示

- **WHEN** 文档被选中时
- **THEN** 该文档的所有问题在右侧栏展示

### 需求：快速导航

点击列表中的问题应滚动对话到该问题位置。

#### 场景：跳转到问题

- **WHEN** 用户点击问题项时
- **THEN** 对话视图滚动到并高亮对应的问题

### 需求：活动问题高亮

对话中当前可见的问题应在列表中高亮。

#### 场景：高亮活动问题

- **WHEN** 用户滚动浏览对话时
- **THEN** 列表中对应的问题被高亮
