# 对话视图规范

## 新增需求

### 需求：对话展示

中间栏应展示包含问题和回答的对话。

#### 场景：对话渲染

- **WHEN** 文档被选中时
- **THEN** 该文档的对话内容在中间栏展示

### 需求：问题对齐

对话中的问题应右对齐并使用独特样式。

#### 场景：问题外观

- **WHEN** 渲染问题消息时
- **THEN** 消息右对齐，使用主题色背景

### 需求：回答对齐

对话中的回答应左对齐并使用独特样式。

#### 场景：回答外观

- **WHEN** 渲染回答消息时
- **THEN** 消息左对齐，使用浅色背景

### 需求：滚动到底部

新消息应自动滚动对话到视图底部。

#### 场景：自动滚动

- **WHEN** 新消息添加到对话时
- **THEN** 视图自动滚动以显示最新消息
