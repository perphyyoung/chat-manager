# 文档列表规范

## 新增需求

### 需求：文档列表展示

左侧栏应展示包含标题的文档列表。

#### 场景：文档列表显示

- **WHEN** 应用加载时
- **THEN** 所有文档在左侧栏展示，包含标题

### 需求：文档选择

点击列表中的文档应选中该文档并更新中间和右侧栏。

#### 场景：选择文档

- **WHEN** 用户点击文档项时
- **THEN** 该文档被高亮为选中状态并更新内容

### 需求：文档导航

选中的文档应保持高亮，直到选择另一个文档。

#### 场景：切换选中项

- **WHEN** 用户选择不同的文档时
- **THEN** 清除之前的选中状态并高亮新文档
