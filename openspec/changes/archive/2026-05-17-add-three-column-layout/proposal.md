# 三栏布局提案

## 为什么

当前应用缺少清晰的文档管理界面布局，用户需要一个直观的三栏布局来高效管理对话式文档。左侧展示文档列表便于导航，中间展示对话内容便于阅读，右侧展示问题列表便于快速定位。

## 变更内容

- 创建三栏布局主界面组件
- 左侧栏：文档列表组件，显示所有文档
- 中间栏：对话式文档展示组件，问题右对齐，回答左对齐
- 右侧栏：当前文档的问题列表组件，支持快速跳转
- 更新主应用布局以集成三栏组件

## 能力

### 新增能力

- `three-column-layout`：三栏布局主界面，包含文档列表、对话展示和问题列表
- `document-list`：左侧文档列表组件，支持文档选择和导航
- `conversation-view`：中间对话展示组件，支持问题右对齐、回答左对齐的聊天样式
- `question-list`：右侧问题列表组件，显示当前文档的所有问题，支持点击跳转

### 修改的能力

无

## 影响

- 修改 App.vue 主布局
- 创建新组件目录结构
- 更新样式文件
- 可能需要更新路由配置
