# 全局搜索功能提案

## 问题

随着文档数量增长，用户难以快速找到特定内容。当前没有跨文档、问题、回答和标签的全文搜索能力，用户只能通过：

1. 手动浏览左侧文档列表
2. 在标签筛选区逐个查找
3. 依赖记忆文档名称

这在有数十个文档时效率极低。

## 解决方案

添加全局搜索功能，支持：

- **多类型搜索**：文档标题、问题文本、回答内容、标签名称
- **模糊匹配**：支持拼写错误容忍和近似搜索
- **即时响应**：输入时实时搜索，无需点击按钮
- **智能跳转**：搜索结果点击后直接定位到对应位置

## 用户体验

### 唤起方式

- **快捷键**：`Ctrl+F` 唤起搜索面板
- **搜索面板**：模态浮层，不离开当前页面

### 搜索面板 UI

```
┌─────────────────────────────────────────────────────────────────┐
│  🔍 [搜索框_____________________________________] [x]           │
├─────────────────────────────────────────────────────────────────┤
│  📄 文档                                    (最多10条)           │
│  ❓ 问题                                    (最多10条)           │
│  💬 回答                                    (最多10条)           │
│  🏷️ 标签                                    (最多10条)           │
├─────────────────────────────────────────────────────────────────┤
│  ↑↓ 导航  Enter 跳转  Esc 关闭                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 交互行为

- **防抖**：输入后 300ms 无新输入才触发搜索
- **导航**：↑↓ 键在结果间切换，Enter 跳转
- **关闭**：Esc 键或点击外部关闭面板

### 跳转行为

| 搜索类型 | 跳转后行为 |
|----------|------------|
| document | 选中该文档，显示对话视图 |
| question | 选中该文档 + 滚动到该问题 + 高亮 |
| answer | 选中该文档 + 滚动到该回答 + 高亮 |
| tag | 左侧标签筛选切换到该标签 |

### 结果展示

- 匹配文本高亮显示
- 显示匹配上下文（snippet）
- 显示关联元数据（如问题所属文档）

## 技术方案

### 搜索技术

- **后端 FTS5**：SQLite 内置全文搜索，支持大规模数据
- **词干提取**：英文词干分析，提高召回率
- **Unicode 支持**：支持中文等多语言搜索

### 数据模型

```sql
CREATE VIRTUAL TABLE search_fts USING fts5(
  id,
  type,        -- 'document' | 'question' | 'answer' | 'tag'
  content,     -- 可搜索文本
  metadata,    -- JSON: { title, parentId, parentTitle, ... }
  tokenize='porter unicode61'
);
```

### IPC 接口

```typescript
ipcMain.handle("search:query", async (_, query: string) => {
  // FTS5 模糊搜索，返回分组结果
});
```

## 范围

### 包含

- 搜索面板 UI 组件
- 后端 FTS5 索引和查询
- IPC 通信层
- 搜索结果高亮渲染
- 键盘导航支持

### 不包含

- 搜索历史（未来功能）
- 搜索结果排序自定义（默认按相关性）
- 正则表达式搜索
- 搜索统计/分析

## 成功标准

1. `Ctrl+F` 可唤起搜索面板
2. 输入关键词后 500ms 内显示搜索结果
3. 支持搜索：文档、问题、回答、标签
4. 点击搜索结果可正确定位
5. ↑↓ 键可导航结果，Enter 跳转
6. Esc 可关闭搜索面板

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| FTS5 索引同步复杂 | 使用触发器自动同步 |
| 大量数据搜索慢 | 限制每类型 10 条结果 |
| 中文搜索支持 | 使用 unicode61 分词器 |
